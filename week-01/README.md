-   커밋 메시지는 아래의 내용으로 통일한다.

```

```

> Q. 처음 저장소를 Clone 하면 모든 파일은 Untracked이면서 Unmodified 상태이다. (Y/N) N

> Q. Git은 순수한 델타 기반 버전 관리 시스템이다. (Y/N) N
